PHP 변수의 종류
    지역변수
        특정 지역내에서만 사용가능한 변수
        함수 내부에서 선언된 변수, 내부에서 선언된 변수는 함수가 종료되면 메모리에서 삭제
        {
            $num = 10; //지역변수
        }
        echo $num;
    전역변수
        함수 외부에서 선언된 변수, 함수 밖에서만 사용할 수 있음
        함수 내부에서 사용하려면 global 키워드를 함께 이용해야 함
        #num = 10;
        {
            gloabal $num;
            echo num;
        }

    정적변수(static)
        함수 내부에서 static 키워드로 선언된 변수
        정적변수는 함수 호출이 종료되어도 메모리 상에서 삭제되지 않음
    
    슈퍼글로벌변수
        내장 변수라고 생각하면 된다.
        미리 정의된 전역변수, 특별한 선언 없이 스크립트 내의 어디에서도 바로 사용할 수 있음
        $_GET, $_PoST, $_COOKIE, $_SESSION

데이터 전송방식(method)
    <form action='목적지' method='방식(get/post)'>

    </form>

get
    URL에 데이터를 포함해서 전송
    URL에 데이터가 노출되어 보안에 취약
    전송하는 길이에 제한이 있음
    속도가 빠름
    캐싱할 수 있음
        캐싱 -> ...
                데이터를 저장하는 방식중 하나
    ex) 네이버 검색

    http://localhost/day02/3_get_ok.php?name=용&query=훈

    ? : get방식의 데이터 전송
    name, query : 전송하는 데이터의 key값(input의 name속성 값)
    & : 두개 이상의 데이터 전달 구분자
    용, 훈 : 전송하는 데이터(input의 value속성 값)

post 
    URL에 데이터를 노출하지 않고 전송
    URL에 데이터가 노출되지 않아 기본 보안은 지켜짐
    전송하는 길이에 제한이 없음
    캐싱할 수 없음
    GET 방식보다 속도가 느림

jQuery(제이쿼리)
    오픈소스 기반의 자바스크립트 라이브러리
    문서객체모델(DOM)과 이벤트(Event)에 관한 처리를 쉽게 구현할 수 있음
    Ajax 응용 프로그램도 빠르게 개발이 가능

    점차 사라진다고 함
        웹 표준화가 안된시절에 어떤 브라우저를 사용하던 잘 작동하게하는 기술
        과거에는 많이 사용중
        현재 60:40 정도 점차 줄어드는 중

라이브러리 vs 프레임워크   
    라이브러리 : 개발하기 위한 도구를 제공
    프레임워크 : 개발하기 좋은 환경을 제공
