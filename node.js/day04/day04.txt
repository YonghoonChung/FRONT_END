JSON(Javascript Object Notation)
    언어가 다른 곳에서 데이터를 교환하고 저장하기 위한 곳
    데이터를 교환하고 저장하기 위해 만들어진 텍스트 기반의 데이터 교환 표준언어
    { Key:Value }
    { 배열명:[값1, 값2, ...]}

JSON 문법 검증
    https://jsonlint.com/

JSON.parse("문자열")
    JSON 포맷으로 되어있는 문자열을 JSON 객체로 변환
